What is Kit?

Kit is a dependency management tool for Objective-C libraries. You declare the projects you depend on, and it will take care of managing them as part of an XCode project.

Installing Kit

Babushka is the recommended way to get started with Kit. Installing babushka and kit is really easy:

$ bash -c "`curl -L babushka.me/up`"
      
(time passes, babushka bootstraps and install itself)

$ babushka sources -a nkpart git://github.com/nkpart/babushka-deps
$ babushka nkpart/kit
