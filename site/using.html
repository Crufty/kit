Consuming Kits

Once you've installed kit, and perhaps got a few kits publishing in your local repository, it's time to start taking advantage of them.

Any project that consumes kits will need a KitSpec file in the root of the project:

$ cat KitSpec
{
      "name": "your-project"
    , "version": "1.0"
    , "dependencies" : [
      { "name": "blocks", "version": "0.9"}
    ]
}

This project, your-project, is at version 1.0 and depends on blocks-0.9.

The next step is to pull in the dependency (or dependencies):

$ kit me
      ... output happens ...
