Command Reference

me

Grabs any dependencies specified in the local KitSpec, and extracts them into Kits.

An XCode project (Kits/KitDeps.xcodeproj) is then generated. It contains all source files specified in the dependencies and will produce a shared library for linking with a parent project.

It will also generate a Kits/KitDeps.xcconfig file. A project that consumes kits should be based on this configuration.

deploy-local

Bundles up this project into a package: dist/[project name]-[project version].tar.gz (where the name and version are read from KitSpec).

The package is then deployed into the local repository: .kit/repository.
